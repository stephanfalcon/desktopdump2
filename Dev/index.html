Position: Full Stack Engineer

Location: 100% Remote

Duration: Full Time/ Perm

As a full-stack engineer you will be responsible for designing, developing, and deploying product features to improve the experience of our users. In any given sprint you might be working directly on an existing customer issue, building something new requested by a customer, or working on scaling and tuning the performance of our platform.

When you get started these are some things you will be working on:

  • Updating and expanding our public-facing API
  • Integrating to external data service to enhance the data in our platform
  • Adding advanced analytics and reporting capabilities to our customer-facing application

Experience you will need:
Ideal candidates will be comfortable doing fullstack work in:

  • Python
  • Java
  • Javascript/Angular
  • Postgres
  • AWS

It is a bonus if they will have worked in a micro-service architecture previously

  • You have couple of years of experience in software development
  • You are excited about working in a small fast-paced environment
  • You are comfortable with some ambiguity and driving work that is not fully defined when you start
  • You have production experience in either Python or Java. Preferably both.
  • You have worked with products deployed to AWS
  • Experience with Javascript and Angular is a bonus
  • Experience with micro-service architecture is a bonus
